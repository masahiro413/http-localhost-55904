# SPEC.md — 個人用シンプル ToDo アプリ 仕様書

## 1. 目的と背景

PC の Web ブラウザ上で動く、個人用のシンプルなタスク管理アプリを作る。
機能は「タスクの**追加・完了・削除**」に絞る。

ただし、この草案の裏にある本当の目的は **「Web の基礎を素の状態で学ぶこと」** である。
アプリ自体は意図的にミニマルに保ち、以下を学習の対象とする。

- HTML / CSS / JavaScript を、フレームワークなしの「素の状態」で書く
- DOM 操作・イベント処理・`localStorage` を直に理解する
- コードの読みやすさと、**毎日開きたくなる見た目（CSS）** を両立させる

## 2. スコープ

### 2.1 対象（やること）

- タスクの追加（1行のテキスト）
- タスクの完了（一覧から即座に消える／数秒間は取り消し可能）
- タスクの削除（完了とは別の操作。実行前に確認を挟む）
- `localStorage` によるデータの永続化（同一ブラウザ内）
- カード型の親しみやすい UI／OS 設定に追従するライト・ダークテーマ

### 2.2 対象外（やらないこと）

- サーバー／アカウント／複数デバイス間の同期
- 締め切り・優先度・カテゴリ・メモなど、テキスト以外の属性
- 完了タスクの履歴・集計・振り返り
- 検索・フィルタ・タグ
- タスクの編集

## 3. データ設計

```js
{
  id: string,
  text: string,
  createdAt: number
}
```

- 保存キーは `simple-todo.tasks`
- 破損データは空配列にフォールバックする

## 4. UI / UX

- タスクはカード型で表示する
- 新規タスクは末尾に追加する
- 完了時は 5 秒間だけ Undo トーストを表示する
- 削除時は確認ダイアログを表示する
- 空状態メッセージを表示する
- キーボード操作と `aria-label` を用意する
