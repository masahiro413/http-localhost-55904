# DESIGN.md

このアプリの見た目は、**カード型・親しみやすい・毎日開きたくなる**ことを目標にする。

## デザイン方針

- 画面全体は中央寄せの 1 カラムにする
- タスクは角丸カードとやわらかい影で表現する
- 余白をゆったり取り、読みやすさを優先する
- ライト / ダークテーマは `prefers-color-scheme` で OS 設定に追従する
- 配色は CSS カスタムプロパティで管理し、テーマ切り替えを分かりやすくする

## UI の要点

- ヘッダーにアプリ名と未完了件数を表示する
- 入力欄と追加ボタンは横並びで、連続入力しやすい見た目にする
- 空状態ではやさしい案内文を表示する
- Undo トーストは画面下部に固定し、すぐに気づけるが邪魔になりすぎない大きさにする
